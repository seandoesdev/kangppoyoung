## 판정 — 되돌림

- **Q4 미해결**: temp 0 인데도 여전히 **3/10 "근거 없음"**(std 40.4). OpenAI는 temp 0 에서도
  잔여 비결정성이 있어 `searchTerms` 가 미세하게 흔들리고, 그 결과 성장단계 funds 청크가 후보에서
  빠지는 표본이 남는다. → **결정론화는 Q4의 근본 해법이 아님**.
- **Q8 회귀(−18)**: temp 0 가 합성을 단답 "예"(15점)로 **고정**. 기존엔 가끔 풀어 답해 33점이었음 →
  Q8 은 **합성 프롬프트** 문제(예/아니오형도 근거를 설명)이지 retrieval/온도 문제가 아님이 확인됨.
- 평균 −1.2(노이즈 범위) + Q8/Q9/Q10 손실 → **순효과 음(−)**. 되돌린다.

## 확정된 다음 타깃

1. **Q4 (retrieval recall)** — funds 청크가 일관 회수되도록: 무정보 청크(표지·목차) 강등 +
   원 질의 안정 회수 패스 병합(searchTerms 변동과 무관한 후보 확보). 온도 아님.
2. **Q8 (synth prompt)** — 예/아니오형 답변에 "결론 + 구체 근거(분야↔품목 연결)"를 강제.
